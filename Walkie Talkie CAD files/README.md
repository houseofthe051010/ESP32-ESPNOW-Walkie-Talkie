# Walkie Talkie CAD Files

This folder is reserved for the open-source mechanical design files for the ESP32 walkie-talkie project, including the custom 3D-printed casing.

Recommended files to place here:

- 3D-printed casing source files
- Enclosure modifications
- Internal brackets
- OLED mounts
- ESP32 mounting plates
- Speaker retainers
- Battery holders
- Antenna mounts
- STL files for printing
- STEP files for editing
- Fusion 360, Onshape, FreeCAD, or other source CAD files

The goal is for someone to be able to build the walkie talkies from the firmware, wiring documentation, photos, bill of materials, and CAD files in this repository.
